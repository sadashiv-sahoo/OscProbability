Detailed Description

OscProbabilitySTD is heart of this whole progamme. Here, "STD" stands for Standard.

For performing the neutrino oscillation, It assigns the values of mass-squared splittings, Mixing Angles, ... etc. One needs to appropriately assign the oscillation parameters and controlling values via the predefined Struct. This can return the oscillation probabilities for Vaccum, Constant matter density, User's Profile density and Earth matter density.
