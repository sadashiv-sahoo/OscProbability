Detailed Description

OscProbability assigns the No. of flavors to perform calculations.

For performing the neutrino oscillation, we need atleast 2 neutrinos to act on. It can calulate the oscillation probabilities in a "n" No. of neutrino Paradigsm.
One needs to appropriately assign the oscillation parameters and controlling values via the predefined Struct.
