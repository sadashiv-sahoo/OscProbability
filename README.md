# OscProbability
## v3.0.0 (Multi Flavour Neutrino Oscillations)

**About:**

  This is an entry-level, Fully-Matrix-Method based Multi Flavour Neutrino Oscillation, C++ Code.
  The Code is User-Friendly and easy to Understand.
