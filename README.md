# OscProbability
## v1.0.0 (Three Flavour Neutrino Oscillations)

About:

  This is an entry-level, Fully-Matrix-Method based Three Flavour Neutrino Oscillation, C++ Code.
  The Code is User-Friendly and easy to Understand.
